Welcome to Library Service

      /api/books

      Get all books
    1. Method: GET
    2. Returns: JSON
    3. Returns a list of all the books in the library, including their details.

      /api/books/{book_id}

    1. Method: GET
    2. Returns: JSON
    3. Returns the details of a specific book based on its book_id.

      /api/books

    1. Method: POST
    2. Returns: JSON
    3. Adds a new book to the library.

      /api/books/{book_id}

    1. Method: PUT
    2. Returns: JSON
    3. Updates the details of a specific book based on its book_id.

      /api/books/{book_id}

    1. Method: DELETE
    2. Returns: JSON
    3. Deletes a specific book based on its book_id.

      /api/books/stats

      Get statistics
    1. Method: GET
    2. Returns: JSON
    3. Returns statistics about the books, including total number of books, how many are read, genre counts, and average rating.
